# Prakhar Enterprises — Business Website

Modern, responsive business website built with **Next.js (App Router)** and **Tailwind CSS**.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Pages

- Home: `/`
- About: `/about`
- Products & Services: `/products-services`
- Solar Solutions: `/solar-solutions`
- Contact: `/contact`
